# Название проекта: NeuroStartUp
Описание: NeuroStartUp - динамически развивающийся стартап, специализирующийся на поиске с использованием новейших технологий искусственного интеллекта.
## Начало работы.
Для создания копии репозитария примените операцию git clone, указав url репозитария. Либо, зайдя в репозитарий на сайте github.com нажмите кнопку "Clone or download"
### Prerequisites
Для использования установите на локальном компьютере Git
